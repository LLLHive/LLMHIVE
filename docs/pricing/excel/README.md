# LLMHive Pricing Tables - Excel Compatible

## How to Use

All files are in **CSV (Comma Separated Values)** format, which can be:
- Opened directly in Microsoft Excel
- Opened in Google Sheets
- Imported into any spreadsheet application

### Opening in Excel
1. Double-click any `.csv` file
2. Excel will automatically open it with proper columns

### Opening in Google Sheets
1. Go to Google Sheets
2. File → Import → Upload
3. Select the CSV file
4. Choose "Replace spreadsheet" or "Insert new sheet"
