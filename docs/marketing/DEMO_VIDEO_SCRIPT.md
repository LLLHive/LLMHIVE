## Recording Tips

### Screen Recording Settings
- Resolution: 1920x1080 or 4K
- Frame rate: 60fps
- Show mouse clicks with visual indicator
- Use a clean browser with no extensions visible

### Audio
- Use professional microphone
- Record in quiet environment
- Consider professional voiceover

### Editing Notes
- Add subtle zoom on important UI elements
- Include text overlays for key points
- Use LLMHive brand colors for graphics
- Add soft background music

### B-Roll Suggestions
- Team working at computers
- Abstract AI/neural network visuals
- Happy customers (if testimonials available)
