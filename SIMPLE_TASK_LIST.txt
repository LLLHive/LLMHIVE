================================================================================
LLMHIVE PROJECT - SIMPLE TASK LIST FOR YOU
================================================================================

Follow these steps in order. Each step is simple and explained clearly.

================================================================================
PART 1: SET UP STRIPE (PAYMENT SYSTEM)
================================================================================

STEP 1: Create Stripe Account
-----------------------------
1. Go to: https://stripe.com
2. Click "Sign up"
3. Enter your email and create password
4. Complete setup
5. Write down your login info

Time: 10 minutes


STEP 2: Get Your Secret Key
----------------------------
1. In Stripe, click "Developers" (left menu)
2. Click "API keys"
3. Find "Secret key" section
4. Click "Reveal test key"
5. Copy the key (starts with sk_test_)
6. Save it somewhere safe

Time: 3 minutes


STEP 3: Set Up Webhook
----------------------
1. In Stripe, go to "Developers" → "Webhooks"
2. Click "Add endpoint"
3. Enter URL: https://your-website.com/api/v1/billing/webhooks/stripe
   (Replace your-website.com with your actual website)
4. Select these events:
   - customer.subscription.created
   - customer.subscription.updated
   - customer.subscription.deleted
   - invoice.payment_succeeded
   - invoice.payment_failed
5. Click "Add endpoint"
6. Copy the "Signing secret" (starts with whsec_)
7. Save it somewhere safe

Time: 5 minutes


STEP 4: Send Keys to Technical Person
--------------------------------------
Send this message to your technical person:

"Hi [Name],

I've set up Stripe. Please add these to the server:

STRIPE_SECRET_KEY = [paste your secret key here]
STRIPE_WEBHOOK_SECRET = [paste your webhook secret here]

Also install: pip install stripe>=7.0.0

Thanks!"

Time: 2 minutes


================================================================================
